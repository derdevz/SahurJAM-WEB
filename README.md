# İFTAR VAKTİ - SahurJAM 2026

Ramazan ayının bereketli atmosferinde geçen bir cooking management oyunu.

## Özellikler
- **Kültürel Otantiklik:** Gerçek Türk mutfağı tarifleri ve Ramazan teması.
- **Dinamik Gün Döngüsü:** İftar Hazırlığı, İftar Rush, Gece Arası ve Sahur fazları.
- **Hurma Ekonomisi:** Kazandığın hurmalarla çadırını geliştir.
- **Müşteri Çeşitliliği:** Farklı sabır ve ödül değerlerine sahip karakterler.

## Teknik Stack
- **Frontend:** React (TypeScript) + Vite
- **Styling:** Tailwind CSS + Radix UI
- **Assets:** Custom Pixel Art Assets

## Kurulum ve Çalıştırma

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Üretim için build al
npm run build
```

## Proje Yapısı
- `src/`: React uygulama kodları.
- `public/asset/`: Oyunun pikselli grafik ve ses varlıkları.
- `src/app/components/`: UI bileşenleri ve oyun mekanikleri bölümleri.
- `src/styles/`: Global temalar ve yazı tipleri.
